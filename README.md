# Python Web Site Crawler

## Overview

Simple Python Code to read the meta data and content from the website url

## Dependencies

This library depends on:

* python 3.6.7
* beautifulsoup4 4.6.3
* newspaper3k 0.2.8

## Setup the Env 

Make sure you already have virtualenv and Python3 

```
bash setup.sh
```

## Run the code

To get started, simply run this command:

```
python3 crawler.py http://google.com
```
